Preferências do sistema

📊 Taxa CDI

Consulte em bcb.gov.br e atualize quando o Banco Central mudar a taxa.

🔄 Cotações Automáticas

Cripto (BTC, USDC...) atualiza sem token via CoinGecko. Para BOVA11, MXRF11 e outros ativos da B3, crie uma conta gratuita em brapi.dev e cole o token acima.

🧩 Módulos

Desative o que você não usa para deixar o app mais limpo. Os dados não são apagados — só ficam ocultos.

📤 Exportar & Backup

O JSON inclui tudo: transações, cartão, investimentos, metas, orçamentos e recorrentes. Use para backup ou migrar para outro dispositivo.
